# Удаление тишины

Автоматически обнаруживает и удаляет участки тишины из клипа. Обнаружение выполняется полностью на устройстве по аудиоволне клипа — интернет-соединение и транскрипция не требуются.

## Рабочий стол

### 1. Выберите клип
Нажмите на любой **видео- или аудиоклип** на временной шкале. Кнопка «Удалить тишину» активируется только при выборе одного клипа (или связанной пары аудио/видео).

### 2. Откройте панель
Нажмите на **значок волны минус** (`waveform.badge.minus`) на панели инструментов или выберите **Edit → Remove Silence**. Панель откроется и сразу начнёт обнаруживать участки тишины с текущими настройками.

### 3. Настройте параметры

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| **Threshold** | −35 dB | Порог громкости, ниже которого звук считается тишиной. Поднимите ближе к 0 dB, чтобы захватывать тихие паузы; опустите, чтобы удалять только почти полную тишину. |
| **Min duration** | 0.5 s | Минимальная длительность тишины, которая будет удалена. |
| **Edge padding** | 0.05 s | Звук, сохраняемый с каждой стороны обнаруженного участка тишины, чтобы речь не обрывалась. |

После изменения любого параметра нажмите **Detect** для повторного обнаружения. Нажмите **Remove Silences** для применения. Нажмите **⌘Z** для отмены.

## Связанные аудио/видеоклипы
При связывании выбор любого клипа выбирает пару. Разрезы выполняются **на обеих дорожках** в одних и тех же кадрах.

## ИИ-агент (MCP)
MCP-сервер по адресу `http://127.0.0.1:19789/mcp`. Используйте инструмент `remove_silence`.

### Инструмент: `remove_silence`
```json
{"name":"remove_silence","arguments":{"clipId":"<id>","thresholdDb":-35,"minSilenceDuration":0.5,"edgePadding":0.05}}
```

#### Параметры
| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `clipId` | string | **обязательный** | Идентификатор клипа из `get_timeline`. |
| `thresholdDb` | number | `−35` | Порог громкости в dBFS (≤ 0). |
| `minSilenceDuration` | number | `0.5` | Минимальная тишина в секундах. |
| `edgePadding` | number | `0.05` | Секунды, сохраняемые с каждой стороны. |

## Советы
- **Ничего не удаляется?** Понизьте порог или уменьшите Min duration.
- **Речь обрывается?** Увеличьте Edge padding.
- **Слишком много разрезов?** Увеличьте Min duration.
- **Связанные клипы рассинхронизированы?** Запустите удаление тишины, пока клипы связаны.
