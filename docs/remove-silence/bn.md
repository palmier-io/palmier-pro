# নীরবতা সরানো

ক্লিপ থেকে নীরব অংশগুলি স্বয়ংক্রিয়ভাবে শনাক্ত করে সরিয়ে ফেলুন। শনাক্তকরণ সম্পূর্ণরূপে ডিভাইসে ক্লিপের অডিও ওয়েভফর্ম থেকে চলে — কোনো ইন্টারনেট সংযোগ বা ট্রান্সক্রিপ্টের প্রয়োজন নেই।

## ডেস্কটপ

### ১. একটি ক্লিপ নির্বাচন করুন
টাইমলাইনে যেকোনো **ভিডিও বা অডিও ক্লিপ**-এ ক্লিক করুন। Remove Silence বোতামটি শুধুমাত্র তখনই সক্রিয় হয় যখন একটি একক ক্লিপ (বা একটি লিঙ্কড অডিও/ভিডিও জুটি) নির্বাচিত থাকে।

### ২. শিটটি খুলুন
টুলবারে **waveform minus icon** (`waveform.badge.minus`) ক্লিক করুন, অথবা **Edit → Remove Silence** বেছে নিন। শিটটি খোলে এবং বর্তমান সেটিংস ব্যবহার করে তাৎক্ষণিকভাবে নীরবতা শনাক্ত করা শুরু করে।

### ৩. প্যারামিটারগুলি সামঞ্জস্য করুন

| প্যারামিটার | ডিফল্ট | বিবরণ |
|-------------|--------|--------|
| **Threshold** | −35 dB | যে লাউডনেস ফ্লোরের নিচে অডিওকে নীরব বলে গণ্য করা হয়। |
| **Min duration** | 0.5 s | সবচেয়ে ছোট নীরবতা যা সরানো হবে। |
| **Edge padding** | 0.05 s | একটি শনাক্তকৃত নীরবতার প্রতিটি পাশে কতটুকু অডিও রাখা হবে। |

### ৪. প্রয়োগ করুন
**Remove Silences** ক্লিক করুন। পূর্বাবস্থায় ফেরাতে **⌘Z** চাপুন।

## লিঙ্কড অডিও/ভিডিও ক্লিপ
লিঙ্কড অবস্থায়, যেকোনো একটি ক্লিপ নির্বাচন করলে জুটিটি নির্বাচিত হয়। Remove Silence **উভয় ট্র্যাক** ঠিক একই ফ্রেমে কাটে।

## AI এজেন্ট (MCP)
`http://127.0.0.1:19789/mcp`-এ MCP সার্ভার। `remove_silence` টুল ব্যবহার করুন।

### Tool: `remove_silence`
```json
{"name":"remove_silence","arguments":{"clipId":"<id>","thresholdDb":-35,"minSilenceDuration":0.5,"edgePadding":0.05}}
```

#### প্যারামিটারসমূহ
| Parameter | Type | Default | বিবরণ |
|-----------|------|---------|--------|
| `clipId` | string | **required** | `get_timeline` থেকে Clip ID। |
| `thresholdDb` | number | `−35` | dBFS-এ লাউডনেস ফ্লোর (≤ 0)। |
| `minSilenceDuration` | number | `0.5` | সেকেন্ডে ন্যূনতম নীরবতা। |
| `edgePadding` | number | `0.05` | প্রতিটি পাশে রাখা সেকেন্ড। |

## টিপস
- **কিছুই সরানো হয়নি?** Threshold কমান অথবা Min duration হ্রাস করুন।
- **বক্তৃতা ছাঁটাই হয়েছে?** Edge padding বাড়ান।
- **অনেক বেশি কাটা?** Min duration বাড়ান।
- **লিঙ্কড ক্লিপ সিঙ্কের বাইরে?** ক্লিপগুলি লিঙ্কড থাকা অবস্থায় Remove Silence চালান।
