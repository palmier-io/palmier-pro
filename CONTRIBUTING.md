# Contributing

## Prerequisites
- macOS 26+
- Xcode 16+
- Swift 6.2 toolchain

## Develop

```bash
git clone https://github.com/palmier-io/palmier-pro
cd palmier-pro

swift build
swift run
```

For a bundled debug build that launches the `.app` and streams OSLog:

```bash
./scripts/dev.sh
```

## Test

```bash
swift test
```
